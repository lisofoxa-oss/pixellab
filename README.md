# 🎨 PixelLab

**Privacy-first image toolkit.** Convert, compress, resize, apply filters, and extract color palettes — all directly in your browser. Nothing is uploaded to any server.

**Live:** [lisofoxa-oss.github.io/pixellab](https://lisofoxa-oss.github.io/pixellab)

## Features

| Tool | Description |
|------|-------------|
| 🔄 Convert | PNG, JPEG, WebP, BMP — convert between formats |
| 📦 Compress | Adjust quality with live size estimation |
| 📐 Resize | Scale with aspect ratio lock, presets |
| 🎨 Filters | Grayscale, sepia, invert, blur, brightness, contrast, saturation |
| 🎭 Palette | Extract dominant colors from any image |

## Why PixelLab?

- **100% private** — all processing happens in your browser
- **No uploads** — your images never leave your device
- **No ads, no tracking, no signup**
- **Works offline** after first load
- **Free forever**
